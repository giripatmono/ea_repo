# Security Standards

## SS-001: Centralized Authentication
Customer-facing and employee-facing applications should use approved centralized authentication where applicable.

## SS-002: Role-Based Access Control
Applications must define role-based access control for protected functions.

## SS-003: Audit Logging
Applications handling PII or financial data must record audit logs for critical actions.

## SS-004: Sensitive Data Protection
PII and financial data must be protected through encryption, access control, and data minimization.
